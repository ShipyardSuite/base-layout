# base-layout
Shared components for Shipyard GCDN Microservices

# Installation
install base-layout with `npm install @shipyardsuite/base-layout`.

# Usage
call base-layout with 
```javascript
const { [COMPONENT_NAME] } = require('@shipyardsuite/base-layout');
```
